# Contributori

## Autori originali (comunità Ninux)

La guida originale è stata scritta collaborativamente dalla comunità [Ninux](https://www.ninux.org/) come documento condiviso su Google Docs. Di seguito i contributori identificabili dal documento e dal contesto della comunità:

| Contributore | Sezioni | Note |
|---|---|---|
| Comunità Ninux Roma | Principi, Bridge, VLAN, Vantaggi, Requisiti | Nucleo teorico della guida |
| Comunità Ninux Roma | AirOS, OpenWRT (LuCI), OLSR, Firewalling | Configurazione ground router principale |
| Contributore MikroTik | RouterOS (Mikrotik) | Procedura Winbox + bridge VLAN |
| Contributore EdgeOS | EdgeOS (Ubiquiti EdgeRouter) | Setup completo EdgeRouter + script OLSR |
| Contributore pfSense | pfSense | Configurazione OLSR, NAT, troubleshooting |
| Contributore Linux Box | Linux Box | Setup VLAN/bridge/OLSR su Linux generico |
| Contributore Policy Routing | Policy Routing | Script ip rule/route per tabelle OLSR |

> **Nota**: I nomi specifici dei contributori non sono riportati nel documento originale su Google Docs.
> Se sei uno degli autori originali e vuoi essere attribuito, apri una issue o una PR.
